# CVIČENÍ 13: OOP

Algoritmizace a programování

## CÍL 6: ODEVZDÁNÍ ÚKOLU

V tomhle cvičení je cílem mít vyplněné všechny úkoly v jednotlivých cílech a odevzdat odkaz na svůj repozitář.

Než budeš odevzdávat, zkontroluj si hlavně:

1. že máš vyplněné všechny úkoly ze cvičení,
2. že jsou změny uložené v Git repozitáři,
3. že je repozitář nahraný na GitHub,
4. že odkaz na repozitář funguje i mimo tvoje přihlášení.

### 6.1 Co má být hotové

Před odevzdáním by měl tvůj repozitář obsahovat alespoň:

- třídu pro biomedicínský signál se základními metodami,
- rozšířenou třídu s metodami `min_value`, `max_value`, `normalize` a `__str__`,
- práci se seznamem objektů (filtrování, hledání maxima, řazení),
- hierarchii dědičných tříd (základní `Signal` + aspoň dvě odvozené),
- průběžně uložené změny pomocí `commit`,
- nahrané změny na GitHub pomocí `push`.

> **💡 Tip:** U OOP úloh se vyplatí dělat commity po jednotlivých třídách nebo po větších krocích návrhu.

### 6.2 Odevzdání do e-learningu

<h2 style="color:#c62828;">❗❗ POVINNÉ ODEVZDÁNÍ</h2>
<ul style="color:#c62828;">
