# CVIČENÍ 5: PRÁCE SE SOUBORY, POWERSHELL A GIT

Algoritmizace a programování

## ÚVOD

V tomhle cvičení se posuneš od „kódu v jednom souboru“ k praktičtější práci, kterou využiješ i v domácích úkolech:

- načteš data ze souborů,
- zpracuješ je pomocí stručných Python konstrukcí,
- spustíš kód z příkazové řádky,
- odevzdáš řešení přes GitHub Classroom.

### Co se v tomto cvičení naučíte?

1. **Tuple (n-tice):**

- proč se hodí neměnitelné datové struktury,
- jak funguje rozbalování hodnot,
- proč `zip(...)` a `enumerate(...)` vrací tuple.

2. **List comprehension:**

- kdy se hodí stručný zápis,
- jak přes něj filtrovat a transformovat data.

3. **Načítání dat ze souborů (`txt`, `csv`, `json`):**

- jak správně otevřít soubor přes `with open(...)`,
- proč řešit `encoding`, `delimiter` a `strip`,
- jak pracovat s cestami k souborům (relativní/absolutní, raw string).

4. **Příkazová řádka (hlavně PowerShell):**

- orientace v adresářích,
- základní příkazy pro každodenní práci,
- spuštění Python skriptu z terminálu.

5. **Git minimum pro GitHub Classroom:**

- bezpečný workflow `clone -> status -> add -> commit -> push` (+ `pull`),
- co zkontrolovat, když odevzdání nefunguje.

### Jak bude vypadat úkol?

Samostatný úkol bude zadaný v GitHub Classroom a bude automaticky testovaný.  
V dokumentaci proto nebude závěrečný ani bonusový úkol.

> **Tip:** V tomhle cvičení je cílem hlavně praktičnost. Nejde o „co nejvíc teorie“, ale o to, abys uměl úkol bezpečně načíst, zpracovat a odevzdat.
