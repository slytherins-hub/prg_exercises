## DOPLNĚK: GITHUB STUDENT DEVELOPER PACK (NEPOVINNÉ)

Tohle je volitelný krok. Pro odevzdávání v GitHub Classroom není nutný.

### K čemu to je?

GitHub Student Developer Pack může dát studentům přístup k vybraným nástrojům a benefitům.

- Přehled: [GitHub Education Pack](https://education.github.com/pack)
- Žádost: [Apply for Student Developer Pack](https://education.github.com/pack/join)

### Co obvykle potřebuješ

- studentský status,
- školní e-mail nebo jiný doklad o studiu,
- základně vyplněný GitHub profil.

### Doporučení

- Schválení žádosti může trvat i několik dní, někdy déle.
- Pro vyplnění žádosti bývá praktické použít telefon (snadněji přiložíš fotky).
- Připrav si předem fotku studentské kartičky/ISIC nebo jiný doklad o studiu.
- Pokud to nestíháš, přeskoč to a vrať se k tomu později.
- Priorita pro cvičení je funkční GitHub účet + nainstalovaný Git.
