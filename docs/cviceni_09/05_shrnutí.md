# CVIČENÍ 9: ZÁKLADY GIT A GITHUB

Algoritmizace a programování

## SHRNUTÍ

V tomto cvičení jsme se naučili:

1. **Nastavit Git**: Nakonfigurovat editor, jméno a email (`git config --global`).
2. **Založit repozitář**: Inicializovat nový repozitář příkazem `git init`.
3. **Sledovat soubory**: Přidávat soubory ke sledování pomocí `git add`.
4. **Vytvářet revize**: Ukládat změny do historie pomocí `git commit`.
5. **Zobrazovat změny**: Používat `git status`, `git diff` a `git show` pro přehled o stavu.
6. **Procházet historii**: Využívat `git log` a grafické rozhraní `gitk`.
7. **Pracovat s větvemi**: Vytvářet (`git branch`), přepínat (`git checkout`) a mazat větve.
8. **Slučovat větve**: Používat `git merge` a řešit konflikty při slučování.
9. **Spolupracovat přes GitHub**: Klonovat repozitáře (`git clone`), vytvářet forky a pull requesty.
10. **Synchronizovat změny**: Nahrávat (`git push`) a stahovat (`git pull`) změny ze vzdáleného repozitáře.
